Frame Info¶

Functionality¶

Give the node graph access to the frame information. Some transport controls have been added since these images were made.

Inputs¶

None

Parameters¶

None

Outputs¶

  • Current Frame,

  • Start Frame,

  • End Frame

  • Evaluate , this generates a value in the range (0.0 …. 1.0), and represents the position of the ‘player head’ in relation to the total amount of frames to play. all wrapped to standard level.

Example of usage¶

Usage with Map Range node to give percentage of current frame range.
